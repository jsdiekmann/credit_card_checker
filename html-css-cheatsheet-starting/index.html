Advanced Selectors

Selector Type Syntax Example Description
Chaining 1st Selector + 2nd Selector p.journal {CSS code} Apply CSS rule only to elements that meet all of the specifications (In our example, only <p> elements with the "journal" class will be affected)
Descendent Combinator 1st Selector + Space + Nested Element .journal li {CSS code} Apply CSS rule only to the specified elements nested within the 1st selector (In our example, only <li> elements that are nested within a "journal" class element will be affected)
Multiple Selectors 1st Selector + , + 2nd Selector p, h1 {CSS code} Independently apply CSS rule to both specified elements (In our example, both <p> and <h1> elements will be affected)
